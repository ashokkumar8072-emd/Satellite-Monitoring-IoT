# 🛰️ Satellite Health Monitoring System using ESP32

## 📖 Project Overview
This project is an IoT-based Satellite Health Monitoring System developed using ESP32 in the Wokwi simulator. The system monitors satellite health parameters such as temperature, humidity, orientation, and motion. The sensor data is displayed on an OLED screen and uploaded to the ThingSpeak cloud platform for remote monitoring.

---

## 🚀 Features

- Real-time Temperature Monitoring
- Humidity Monitoring
- Roll and Pitch Detection using MPU6050
- OLED Display for Live Data
- Buzzer Alert System
- Cloud Monitoring using ThingSpeak
- ESP32 Wi-Fi Connectivity
- Wokwi Simulation

---

## 🛠 Components Used

- ESP32 Development Board
- DHT22 Temperature & Humidity Sensor
- MPU6050 Accelerometer & Gyroscope
- SSD1306 OLED Display
- Buzzer
- ThingSpeak Cloud Platform

---

## 📊 Cloud Dashboard

Sensor data is uploaded to ThingSpeak for real-time visualization and monitoring.

Parameters:
- Temperature
- Humidity
- Roll Angle

- Health Status
